app.launch source-map verification

Companion to _test-debugger.html. That fixture verifies inline scripts compiled by processRipScripts(); this one verifies the bundle path used by app.launch() — every lazily-compiled component gets an inline source map keyed to its .rip path.

Open DevTools → Console for the diagnostic dump. Open DevTools → Sources panel — files like components/counter.rip should appear as navigable virtual files showing original Rip source.
